Description

Basic wrapper for rsite to put in a new or update an existing freesite from a directory on the same server as the node.

Usage

putsite.rb [OPTIONS] … DIR

-h, —help: 

    This message

-n, —name: 

    The site name and the file to save the rsite info to. If this is found
    in ./ then the details will be loaded from it and the site updated

-t, —type: 

    The type of site, either USK or SSK

Required files

rsite   getoptlong   rdoc/usage
